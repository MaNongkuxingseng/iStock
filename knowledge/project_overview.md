# iStock 项目概述

## 项目简介
iStock是一个智能股票分析系统，提供实时盯盘、技术分析、投资组合管理和自动化交易建议。

## 核心功能
1. **实时市场监控** - 9个时间点的盯盘消息推送
2. **智能分析** - 基于机器学习的技术分析
3. **投资组合管理** - 个人持仓管理和分析
4. **风险预警** - 实时风险检测和警报
5. **数据可视化** - 交互式图表和报表
6. **自动化交易** - 基于策略的自动化建议

## 技术栈
- **后端**: Python + FastAPI + PostgreSQL + Redis
- **前端**: React + TypeScript + Ant Design
- **部署**: Docker + Nginx + Docker Compose
- **监控**: 自定义监控系统 + Grafana
- **CI/CD**: GitHub Actions

## 架构设计

### 微服务架构
```
用户界面 (React) → API网关 (Nginx) → 后端服务 (FastAPI)
                                      ↓
                              数据库 (PostgreSQL)
                                      ↓
                              缓存 (Redis)
                                      ↓
                              消息队列 (Celery)
```

### 数据流
1. 市场数据采集 → 数据清洗 → 特征工程
2. 机器学习模型训练 → 预测分析 → 结果存储
3. 用户请求 → 业务逻辑处理 → 数据查询 → 响应返回
